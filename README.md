# VainZy 🎲

**VainZy** est une implémentation numérique du célèbre jeu de dés **Yatzi** (ou Yahtzee), développée dans le cadre du cours de **Programmation** à la **Haute École de la Ville de Liège (HEL)**.

Ce projet a été conçu pour mettre en pratique les concepts de programmation orientée objet et de logique algorithmique en C#.

## 🛠️ Technologies utilisées

* **Langage** : C#
* **Environnement** : Visual Studio 2022 / .NET Core

---

## 📦 Installation et Utilisation

1. **Cloner le dépôt** :
```bash
git clone https://github.com/votre-utilisateur/VainZy.git

```


2. **Ouvrir le projet** :
Double-cliquez sur le fichier `VainZy.sln` pour l'ouvrir dans **Visual Studio**.
3. **Compiler et lancer** :
Appuyez sur `F5` ou cliquez sur le bouton **Démarrer** dans Visual Studio.
